# Career Hub Consultancy (Static Single Page)

## How to run
1) Unzip the project  
2) Open `index.html` in your browser (double click)

## Edit pamphlet images
Replace these files (or add your own):
`assets/pamphlets/pamphlet-01.png` ... `pamphlet-14.png`

Then update the `<img src="...">` paths in `index.html` if needed.

## Edit contact details
Open `index.html` and update:
- Office address
- Phone/WhatsApp
- Email
